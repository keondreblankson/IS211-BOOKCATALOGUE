# IS211-BOOKCATALOGUE


This project is a Flask web application that allows users to keep track of books they own.

Users can:

- Login
- Search for books by ISBN using the Google Books API
- Save books to a database
- Delete books from their collection

The application stores:

- Title
- Author
- Page Count
- Average Rating
- Thumbnail Image
